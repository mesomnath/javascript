The DOM is a programming interface for HTML documents. JavaScript can manipulate the DOM to create dynamic web pages.

- `getElementById()`
- `getElementsByClassName()`

## Modifying Elements
- `innerHTML`, `textContent`, `innerText`
- `setAttribute()`, `getAttribute()`
- `classList.add()`, `classList.remove()`, `classList.toggle()`
- `style` property

## Creating Elements
- `createElement()`
- `appendChild()`, `append()`
- `insertBefore()`, `insertAdjacentElement()`
- `cloneNode()`

## Removing Elements
- `remove()`
- `removeChild()`

## DOM Traversal
- `parentNode`, `parentElement`
- `children`, `childNodes`
- `firstChild`, `lastChild`
- `nextSibling`, `previousSibling`

## Practical Examples
- Dynamic content updates
- Form validation
- Interactive menus
- Modal dialogs

## Interview Questions

**Q: What is the DOM?**
A: A tree-like representation of HTML document that JavaScript can manipulate.

**Q: Difference between innerHTML and textContent?**
A: innerHTML parses HTML, textContent returns plain text only.

**Q: What is querySelector?**
A: Modern method to select elements using CSS selectors.
